# 🧭 Orientations Stratégiques v4 — Heryze (Point d'Étape)

Ce document résume les points critiques identifiés pour débloquer le développement et assurer la cohérence entre les versions de la base de données.

---

## 1. 🔑 Résolution des Problèmes d'Inscription (Checklist Supabase)

Si l'inscription échoue malgré des clés `.env` correctes, vérifiez ces paramètres dans le dashboard Supabase :

*   **Confirmation d'Email (Le coupable n°1)** : Par défaut, Supabase exige un clic sur un lien mail. En local, cela bloque tout.
    *   **Solution** : `Authentication > Settings > Email Auth` -> Décochez **"Confirm email"**.
*   **Site URL & Redirects** : Vérifiez que the "Site URL" est bien `http://localhost:5173`.
*   **Accès Collaborateur** : Invitez vos collègues via `Settings > Members` pour qu'ils accèdent au dashboard, même si leurs clés `.env` suffisent pour le code.

---

## 🏗️ 2. Unification des Bases de Données (SQL)

Le schéma `supabase/schema.sql` (version du collaborateur) est la référence à adopter car il est plus complet pour un modèle SaaS.

### Ajouts essentiels de la nouvelle version :
- **Table `subscriptions`** : Indispensable pour la future facturation Stripe.
- **Table `staff`** : Gestion des employés (caissiers) avec droits restreints.
- **Trigger `updated_at`** : Traçabilité des modifications de stock.
- **Numérotation des Reçus** : Automatisation légale des numéros de vente (`REC-YYYYMMDD-XXXXX`).

> [!TIP]
> **Conseil** : Adoptez ce fichier SQL. Videz vos tables actuelles avant de relancer le script pour éviter les conflits d'IDs.

## 🧲 4. Fonctionnalités "Aimants à Clients" (Priorités)

Pour séduire rapidement les TPE, voici les fonctions à implémenter en priorité :

1.  **📊 L'Export Comptable** : Un bouton "Magique" générant un CSV/Excel prêt pour le comptable.
2.  **⭐ Gestion des Favoris** : Pouvoir épingler les 5 produits les plus vendus pour un encaissement express.
3.  **📱 Ticket Digital (QR Code)** : Permettre au client de scanner un code pour récupérer son ticket PDF.
4.  **🔄 Multi-terminaux** : Démontrer la synchro temps réel entre un iPhone et un iPad.
